## Project Summary
Coffee House is a full-stack web application for a modern coffee shop, designed as a portfolio project to showcase advanced web development skills. The project features a stylish, responsive frontend, a robust backend with RESTful API, and a SQLite database. It demonstrates best practices in UI/UX, React, Node.js, and deployment.

---

## Features

### Frontend (React + Vite)
- Modern, animated landing page (Hero section, parallax, gradients)
- Dynamic menu with product categories (coffee, desserts)
- Product cards with images, descriptions, and prices
- Category filtering and popular products section
- Contact and About pages
- Responsive design for all devices
- Custom icons and branding
- Footer with contact info, social links, and creator credit

### Backend (Node.js + Express)
- RESTful API for menu, orders, users, and stats
- SQLite database for menu items, users, and orders
- Authentication and admin panel (admin folder)
- Email notifications via SendGrid
- Secure environment variable management
- Middleware for authentication and error handling

### Database
- SQLite database (`coffee-shop.db`) with menu, users, and orders
- Menu items include name, description, price, category, and image path
- Data seeding from `menu.js`

### Deployment
- Frontend deployed on Vercel
- Backend deployed on Render
- Static assets (images) in `frontend/public`
